# iamizana.github.io

Izana's personal site — a small, honestly-maintained corner of the web.

- `index.html` — home, with a live "Currently" line
- `history.html` — every "Currently" line, in order
- `notes.html` — longer-form thoughts
- `currently.json` / `notes.json` — the data the pages read at load time

Updated automatically a few times a day, and by hand whenever something's worth saying.
